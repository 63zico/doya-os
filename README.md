# DOYA OS

DOYA OS is an AI Restaurant Operating System for running, improving, and scaling restaurant operations.

The project is organized as a documentation-first architecture scaffold. It separates business vision, product thinking, operations, user experience, data, engineering, AI, API design, prompts, tests, and architectural decisions before application code is introduced.

## Project Structure

- `docs/` - Product, operation, design, technical, AI, API, testing, and decision documentation.
- `frontend/` - Future frontend application workspace.
- `backend/` - Future backend application workspace.
- `ai/` - Future AI workflows, agents, models, and evaluation assets.
- `database/` - Future schema, migrations, seeds, and data documentation.
- `assets/` - Future brand, design, media, and static assets.

## Principle

DOYA OS should help restaurant teams make better decisions, reduce repetitive work, and connect daily operations with AI-assisted insight.
